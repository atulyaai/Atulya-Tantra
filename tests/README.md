# 🧪 Tests

Automated tests for Atulya Tantra.

## 📁 Structure

```
tests/
├── unit/                 # Unit tests
├── integration/          # Integration tests
├── e2e/                  # End-to-end tests
└── fixtures/             # Test data
```

## 🚀 Running Tests

```bash
# Install test dependencies
pip install pytest pytest-cov pytest-asyncio

# Run all tests
pytest

# Run with coverage
pytest --cov=server --cov=config

# Run specific test
pytest tests/unit/test_sentiment.py
```
