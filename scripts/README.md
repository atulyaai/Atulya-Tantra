# 🔧 Scripts

Utility scripts for Atulya Tantra.

## 📜 Available Scripts

### Setup
- **`setup.sh`** - Linux/Mac setup script
- **`setup.ps1`** - Windows PowerShell setup script

### Testing
- **`test_models.py`** - Compare model performance

### Utilities
- **`clean_data.sh`** - Clear all user data (coming soon)
- **`backup.sh`** - Backup conversations (coming soon)

## 🚀 Usage

### Linux/Mac
```bash
chmod +x scripts/setup.sh
./scripts/setup.sh
```

### Windows
```powershell
.\scripts\setup.ps1
```

### Test Models
```bash
python scripts/test_models.py
```
