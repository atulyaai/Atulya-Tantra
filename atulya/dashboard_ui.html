Glossary & Terminology

MBS: Micro-Batch Size. The subset of training data processed in parallel.

Nodes: Active Neurons. Out of the total model capacity, only a tiny percentage of specific MoE expert neurons actually fire during any given task.

Tok/s: Tokens per second (System Throughput).

Active / Total Params: NP-DNA models store a massive theoretical parameter count (Total), but dynamically activate only the required neural strands (Active) per inference. This allows CPU-bound inference by vastly reducing memory overhead while retaining the knowledge of a huge model.
